Data export

The data are exported into a text directly readable by MS-Excel or by any other spreadsheet, data processing software, ...

One exported file can contain the data related to one or more curves.

Discrete curve : the curve points are written in the order in which they have been defined by the user during the curve construction.

Continuous curve : the points are written in the ascending order of their abscissa value. The first point of the curve is specified by the user, in order to get a round-off number.

The period between each point is defined by the user too. If the abscissa axis is logarithmic, this period is express by the mean of unit decade.

The last written point is the last defined one.

If the curve is not defined on some values (or if the first points specified by the user are not defined), the undefined points are expressed only by their abscissa value. The ordinate value is empty.

The ordinate value of each point is the linear interpolation of the physical points of the curve (see figure).

 

Splines curves : The user chooses the number of points to be saved for each segment of the spline curve.

Densitometry curves : The points are saved according to the direction defined by the user : increasing or decreasing abscissa or ordinate value.

As for the continuous curves, the interval between points is defined by the user, and the values of the saved points are calculated through a linear interpolation of the pixel values on the graphic.

 

Notice : The data can also be saved into a CurveUnscan specific file format (CurveUnscan document). These files contain all the informations related to a document and can be used to reuse the same document (without performing all the recognition process) to export the data in an other format.
