Curves

CurveUnscan handles four kind of curves : discrete curves (scattered points), continuous curves, spline curves and densitometry curves.

Each curve is related to a coordinates system which must have been built previously. Note that an unlimited number of curves can be built on a coordinates system.

Discrete Curve

The discrete curves (scattered points) are composed from a set of independent points. These points are individually identified by the user.

Continuous Curve

The continuous curves are composed with one or more groups of juxtaposed points. A continuous curve is identified through 3 main steps:

  • Identification of the definition range
  • Automatic recognition step, from one ore more genesis points
  • Manual correction of the imperfections that can subsist.

The automatic recognition step identifies the curve from initial points, by following the graphic points. The process can differentiate several crossed curves, can identify dashed curves and takes into account the curve thickness.

Though the coordinates system is totally independent from the physical one (pixels), the continuous curves present some constraints with the coordinates system:

  • The curves are applications from X space to Y space in the physical coordinates system (only one y value is defined for a particular x value)
  • During the data export, the curves are considered as application in the logical coordinates system (the one built by the user)

These constraints have few influence on the curve identification if the coordinates system tilt is less than the largest rate value of the curve. In the case of curves with very sharp peaks, you must take care of the scanning operation in order to have a small graphic tilt.

Spline curves

The spline curves are built using a mathematic definition, from anchor points which position are defined by the user.

With the spline curves, the user is allowed (if the curve is not too much complicated), to quickly interpolate its graphic curve, with a good precision from a limited number of anchor points.

The spline curves are built as in a drawing tool, by clicking on the graphic in order to set and drag the anchor points.

Densitometry curves

The densitometry curves are used to identify a color profil on the graphic. The user defines a segment on the graphic and a value is calculated for each pixel of this segment. This value is calculated according to a color scale which is also defined by the user.
