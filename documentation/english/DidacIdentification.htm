Curve identification

Initial points

In order to identify the curve, CurveUnscan needs to know its location on the graphic. When this dialog box is displayed, the "Pick curve" button is pressed and the dialog box is waiting you to click on the curve.

As the curve may be composed of several parts, you have to click on each of them.

For instance, for the following graphic, if you want to identify the curve A, you can do the following steps :

  1. Click on the segment 1 of the curve A : if the followed side is the top one (see later), CurveUnscan will follow the segment 1 of the curve A, the segment 2 of the curve B and the segment 3 of the curve A.
  2. Click on the segment 2 of the curve A : the segments 1, 2 and 3 of the curve are identified.
  3. In order to identify the right part of the curve, click on the segments 4 and 5.

N.B. : you can obtain the same result by clicking on the segments 2 and 5.

Side to follow

You can specify which side of the curve CurveUnscan will follow : the top or the bottom. In the case of a curve containing positive peaks, following the top lead to a better result. In the case of negative peaks, following the bottom is better.

Curve thickness

The scanned curve having some thickness, the identified curve must be translated in order to take this thickness into account. The thickness value can be defined either manually or by clicking on the "Pick" button then on the curve. The calculated value is the vertical thickness of the curve on this location.

Dashed curve

CurveUnscan can identify dashed curves. When this option is activated, the curve following is not stopped at the end of each dash but is propagated to the next dashes. The distance between dashes must be less than the specified value. In order to change this value, click on the "modify" button.

Curve name

This name will be used in the output file in order to identify this curve.

Advanced options

This button opens a dialog box which allows to modify some advanced parameters : the threshold color for curve identification and the algorithm used for the thickness compensation.
