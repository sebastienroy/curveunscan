CurveUnscan

 

CurveUnscan is a curve recognition software. From a graphic file resulting of the scan of a curve (initially available on paper support), it can identify the points coordinates of this curve. The digitized data are exported into a text file which is formatted in order to be directly readable with any data processing software, MS-Excel for instance.

Main application fields of CurveUnscan

  • Comparison with data published into the literature : after digitalization of curves published into the literature (presented into several graphics in different formats), you can directly compare these data (and also your own data) on the same graphic.
  • Recovery of lost data, from their paper plot.
  • Complementary use with an Optical Character Recognition (OCR) software, in order to achieve the digitalization of your documents.

Even though CurveUnscan is easy to use, reading its working principles can help you to get the best from it.
