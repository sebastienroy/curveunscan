Courbe discrète

Pour identifier un point de la courbe, cliquez simplement sur l'emplacement de celui-ci sur le graphique. Ses coordonnées dans le système de coordonnées défini précédemment apparaissent alors dans la liste de la boite de dialogue. Il est possible de trier ces points dans l'ordre croissant de leur valeur d'abscisse. Il est également possible d'effacer les points sélectionnés de cette liste (le bouton "Effacer" est actif lorsque des points sont sélectionnés).

Le nom de la courbe est utilisé dans le fichier de sortie pour personnaliser les données correspondantes.
