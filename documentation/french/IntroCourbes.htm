Courbes

CurveUnscan permet d'identifier quatre types de courbes : les courbes discrètes (de type nuage de points), les courbes continues, les courbes splines et les courbes de densitométrie.

Une courbe est définie par rapport à un système de coordonnées qui doit avoir été identifié préalablement. Sur un système de coordonnées peuvent être définies un nombre illimité de courbes, de type nuage de points ou de type continue.

Courbe discrète

Les courbes discrètes, ou nuage de points, sont composées d'une succession de points sans lien entre eux. Ces points sont individuellement identifiés par l'utilisateur.

Courbe continue

Les courbes continues sont constituées d'un ou plusieurs groupes de points juxtaposés. Une courbe continue est identifiée en 3 principales étapes :

  • Identification de l'intervalle dans lequel elle est définie
  • Reconnaissance automatique, à partir d'un ou plusieurs points initiaux
  • Correction des imperfections

L'étape de reconnaissance automatique permet la reconnaissance d'une courbe à partir de points initiaux, par suivi des points du graphiques. Elle est capable de différentier plusieurs courbes se croisant, d'identifier une courbe en pointillés et tient compte de l'épaisseur de trait du graphique original. L'étape de correction finale permet de corriger les imperfections de la reconnaissance automatique.

Bien que le système de coordonnées logique soit totalement indépendant du système de coordonnées physique (pixels), les courbes continues présentent certaines contraintes par rapport au système de coordonnées physique :

  • A la construction, les courbes sont des applications de l'espace des x vers l'espace des y dans le système de coordonnées physique (seul un point y est défini pour une valeur de x donnée).
  • A l'enregistrement, les courbes sont également considérées comme étant des applications dans le système de coordonnées logique (défini par l'utilisateur).

Ces contraintes n'ont cependant aucune incidence sur la construction de la courbe si l'inclinaison du système de coordonnées est inférieure à la pente maximale de la courbe. Dans le cas de courbes dont la dérivée prend des valeurs élevées (cas par exemple de courbe comportant des pics de faible largeur), il est nécessaire de veiller à ce que l'inclinaison du système de coordonnées soit faible.

Courbes splines

Les courbes splines sont des courbes construites selon une formule mathématique à partir de "points d'ancrage" définis par l'utilisateur.

Les courbes splines permettent, si la courbe à numériser n'est pas trop complexe, d'interpoler rapidement la courbe à partir d'un nombre limité de points et avec une relativement bonne précision. Elle permettent également de prendre en compte les "courbes paramétrées", comme par exemple les courbes qui bouclent sur elle-mêmes.

Les courbes splines se construisent comme dans un logiciel de dessin, en cliquant sur le graphique et en faisant glisser la souris pour placer les points d'ancrage.

Courbes de densitométrie

Les courbes de densitométrie permettent d'identifier un profil de couleur sur le graphique. L'utilisateur défini un segment sur le graphique et une valeur est calculée pour chaque pixel de ce segment. Cette valeur est calculée par rapport à une échelle de couleur également définie par l'utilisateur.
