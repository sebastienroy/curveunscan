Format du fichier d'export

Cette boite de dialogue vous permet de choisir le caractère qui servira de séparateur de colonne dans le fichier exporté.

Dans les fichiers d'export, les données de chaque courbe, ainsi que leurs valeurs d'abscisse et d'ordonnée, sont séparées par un caractère spécial. L'utilisation d'une tabulation permet par exemple d'ouvrir ces fichiers directement avec MS-Excel.

Le caractère utilisé par défaut peut être modifié dans les Options (Affichage / Options / Export).
