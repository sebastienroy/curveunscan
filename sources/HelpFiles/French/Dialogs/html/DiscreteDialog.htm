Choix des points

Pour identifier les points de la courbe, vous devez cliquer sur chacun d'eux. Leurs coordonnées apparaissent alors dans la liste "Choix des points" dans l'ordre chronologique de leur sélection.

Il est possible de trier ces points dans l'ordre croissant de leur valeur d'abscisse par l'intermédiaire du bouton "Trier".

Pour effacer un ou plusieurs points, sélectionnez les dans la liste et cliquez sur le bouton "Effacer".
