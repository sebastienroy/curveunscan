Options : Export

Cet onglet vous permet de définir le format par défaut des fichiers d'export.

Séparateur de colonne

Dans les fichiers d'export, les données de chaque courbe, ainsi que leurs valeurs d'abscisse et d'ordonnée, sont séparées par un caractère spécial. L'utilisation d'une tabulation permet par exemple d'ouvrir ces fichiers directement avec MS-Excel.

Extension par défaut

L'extension du nom de fichier n'a pas d'influence sur son format. Cette extension permet cependant à certains logiciels d'en déduire le format utilisé et de l'ouvrir de la manière appropriée.
