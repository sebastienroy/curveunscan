Options : Courbe de densitométrie

Cette onglet vous permet de définir les valeur par défaut de l'échelle de couleur utilisée pour les courbes de densitométrie.

Remarque : il est également possible de fixer ces valeurs par défaut depuis la boite de dialogue d'identification des courbes.

 

Une valeur est associé à chacune des couleurs extrêmes de l'échelle. Lors de l'identification des courbes, les valeurs intermédiaires sont calculées linéairement par projection (dans l'espace mathématique des couleurs) sur l'axe reliant ces deux couleurs.

Plus simplement, si l'une des couleurs est le noir et l'autre le blanc, les valeurs calculées pour les couleurs rencontrés correspondent à sa luminosité.

Exemples:

Couleur 1 Valeur 1 Couleur 2 Valeur 2 Pixel Valeur associée
(255,255,255) blanc 0 (0,0,0) noir 100 (127,127,127) gris 50
(0,0,0) noir 0 (255,0,0) rouge 100 (0,255,0) vert

0

(il n'y a pas de composante verte dans le rouge)

Pour modifier les couleurs de l'échelle, cliquez sur le bouton de couleur (une boite de dialogue apparaît, permettant de définir chacune de ses composantes).
