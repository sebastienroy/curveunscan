Type de courbe

Vous devez maintenant choisir quel type de reconnaissance de courbe vous voulez utiliser : courbe continue, courbe spline, courbe discrète (nuage de points), ou courbe de densitométrie.

Courbe continue

Ce mode de reconnaissance permet d'identifier automatiquement, en quelques clics, les courbes continues les plus complexes. Ce mode de reconnaissance gère les courbes qui se croisent, les courbes en pointillés, tiens compte de l'épaisseur du trait...

Courbe spline

Les courbes splines permettent d'identifier très rapidement les courbes du graphique, et notamment les courbes "paramétrées". L'identification se fait en placant à la souris les points d'ancrage des courbes splines, tout comme dans un logiciel de dessin vectoriel.

Courbe discrète (nuage de points)

Ce mode permet d'identifier manuellement des points séparés.

Courbe de densitométrie

Les courbes de densitométrie permettent de quantifier les niveaux de gris de votre graphique. Ce mode vous permet de faire des analyses quantitatives de chromatographies sur plaque par exemple.
