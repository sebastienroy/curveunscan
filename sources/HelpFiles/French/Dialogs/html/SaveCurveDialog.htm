Enregistrement

Avant d'enregistrer les résultats dans un fichier, il est nécessaire de mettre en forme les données précédemment identifiées.

Les noms des courbes qui ont été précédemment identifiées figurent initialement dans la liste de gauche de la boite de dialogue. Pour que les données d'une courbe soient exportées, il suffit de la faire passer dans la liste de droite au moyen du bouton flèche correspondant. Les deux boutons situés à droite de la boite de dialogue permettent de faire varier l'ordre dans lequel ces courbes apparaîtront dans le fichier. Le bouton "Prévisualisation" permet d'afficher à l'écran le résultat final. Le bouton "Format" permet de choisir le caractère de séparation des colonnes dans le fichier d'export.

Pour les courbes continues, les courbes splines et les courbes de densitométrie, une boite de dialogue apparait lorsqu'on fait passer la courbe de la liste de gauche à celle de droite. Cette boite de dialogue permet de spécifier la quantité de points à enregistrer dans le fichier final.
