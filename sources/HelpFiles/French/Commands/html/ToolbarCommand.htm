Barre d'outils

Vous pouvez afficher ou cacher la barre d'outils via le menu Affichage / Barre d'outils

La barre d'outils contient les éléments suivants :

Ces outils correspondent aux commandes suivantes :

  • Fichier / Ouvrir un graphique
  • Fichier / Exporter les données
  • Identification / Nouvelle courbe / Courbe continue
  • Identification / Nouvelle courbe / Courbe discrète
  • Identification / Les deux (nouveau système de coordonnées et nouvelle courbe)
  • Identification / Nouveau système de coordonnées
  • Aide / A propos de CurveUnscan

Le dernier outil permet d'afficher l'aide en ligne sur les différentes commandes.
