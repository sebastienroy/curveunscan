Enregistrer un document

Cette commande est accessible par le menu Fichier / Enregistrer le document. Elle permet d'enregistrer le document au format CurveUnscan

Les documents CurveUnscan contiennent un graphique et tous les éléments nécessaires à l'identification de courbes : axes, points d'origine, nom des courbes...

Le fichier généré par cette commande ne peut pas être traité par d'autres logiciels, contrairement au fichier généré par la commande "Exporter les données". Par contre, enregistrer le document permet de le rouvrir ultérieurement dans CurveUnscan afin par exemple d'affiner une reconnaissance de courbe. Il ne vous est plus nécessaire de reprendre depuis le début toutes les étapes d'identification.
