Zoom

Le facteur de zoom du graphique peut être modifié via le menu Affichage / Zoom

Une fenêtre contenant une liste déroulante permet de choisir le facteur de zoom, entre 10% et 400%. Un élément "Ajuster" est également disponible. Si cet élément est choisi, le facteur de zoom est calculé afin que le graphique soit entièrement contenu dans la fenêtre.
