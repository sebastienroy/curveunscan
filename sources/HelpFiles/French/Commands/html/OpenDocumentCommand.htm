Ouvrir un document

Cette commande permet d'ouvrir un document CurveUnscan. Elle est accessible par le menu : Fichier / Ouvrir un document.

Les documents CurveUnscan contiennent un graphique et tous les éléments nécessaires à l'identification de courbes : axes, points d'origine, nom des courbes... Le fichier document est généré par la commande Fichier / Enregistrer le document.

Le fichier généré par cette commande ne peut pas être traité par d'autres logiciels, contrairement au fichier généré par la commande "Exporter les données". Par contre, enregistrer le document permet de le rouvrir dans CurveUnscan afin par exemple d'affiner par la suite une reconnaissance de courbe. Il ne vous est pas nécessaire ainsi de reprendre depuis le début toutes les étapes d'identification.
