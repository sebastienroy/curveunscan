CurveUnscan

 

CurveUnscan est un logiciel de reconnaissance de graphiques. Il permet, à partir d'un fichier graphique résultant du scannage d'une courbe (disponible initialement sur un support papier), d'identifier les coordonnées des points de cette courbe. Les données ainsi numérisées sont exportées dans un fichier texte qui est formaté afin d'être directement exploitable avec n'importe quel tableur, MS-Excel par exemple.

Principaux champs d'application de CurveUnscan

  • Comparaison de données publiées dans la littérature : après numérisation des coordonnées de courbes publiés dans la littérature (présentés sur des graphiques séparés et dans des formats différents), il est possible de comparer ces données directement sur le même graphique, éventuellement avec ses propres données.
  • Récupération de données de mesure égarées, à partir de la trace papier d'un graphique
  • Utilisation complémentaire complémentaire à un logiciel de reconnaissance de texte (OCR) afin de compléter la numérisation des documents.
 

Bien que CurveUnscan soit relativement simple d'emploi, la lecture des princes de fonctionnement du logiciel est recommandée afin d'en tirer le meilleur parti.
