Export des données

 

Les données sont exportées dans un fichier texte dont le format permet la relecture directement par MS-Excel ou par tout autre tableur ou logiciel de traitement de données. (Les lignes sont séparées par le caractère "retour chariot", les colonnes sont séparées par un caractère, par exemple "Tabulation")

Un fichier peut contenir les données relatives à une ou plusieurs courbes.

Courbe discrète : les points de la courbe sont enregistrés dans l'ordre dans lequel ils ont été définis lors de leur construction.

Courbe continue : les points sont enregistrés dans l'ordre croissant de leur valeur d'abscisse. Le premier point de la courbe est spécifié par l'utilisateur.

La période entre chaque point est également définie par l'utilisateur. Si l'axe des abscisses est logarithmique, cette période est exprimée en décade d'unité.

Le dernier point enregistré est le dernier point défini.

Si la courbe est discontinue (ou si les premiers points spécifiés par l’utilisateur ne sont pas définis), les points non définis sont exprimés uniquement par leur abscisse, le champ correspondant à la valeur d’ordonnées est vide.

La valeur d’ordonnée de chaque point enregistré correspond à l’interpolation linéaire des valeurs d’ordonnée des points de la courbe physique.

Courbes splines : l'utilisateur choisit le nombre de points qui seront enregistrés pour chaque segment de la courbe.

Courbes de densitométrie : Les points sont enregistrés dans le sens défini par l'utilisateur : valeur d'abscisse ou d'ordonnée croissante ou décroissante.

Tout comme pour les courbes continues, l'intervalle entre ces points est défini par l'utilisateur et les valeurs des points enregistrés sont calculées par interpolation linéaire des valeurs calculés des pixels du graphique.

 

Remarque : les données peuvent également être enregistrées dans un format spécifique à CurveUnscan (document CurveUnscan). Ces fichiers contiennent l'ensemble des information relatives au document et permettent de les conserver afin par exemple d'en exporter les données ultérieurement dans un format différent.
