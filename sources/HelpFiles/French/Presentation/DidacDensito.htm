Courbe de densitométrie

Ce type de courbe correspond à l'analyse d'un dégradé de couleur sur le graphique. A chaque pixel d'un segment (que vous définissez), est associée un valeur. Cette valeur est calculée en fonction de la couleur du pixel et d'une échelle de couleur (partie droite de la boite de dialogue).

Tracé du segment

Lorsque le segment à analyser n'a pas encore été défini, le bouton "Tracer le segment" de la boite de dialogue est enfoncé. Tracez alors un segment sur la partie à analyser du graphique. Si ce segment déborde de la fenêtre, le graphique défile automatiquement. Lorsque la touche "Majuscule" est enfoncée, le trait est tracé parallèlement au bord de la fenêtre.

Prévisualisation

Lorsque le segment a été défini, le bouton "Privisualisation" permet de faire apparaître le profil de valeur sur ce segment. La direction de ce profil va du début du segment (partie gauche) tel qu'il a été tracé, à la fin du segment. Cet ordre n'est cependant utilisé que pour la prévisualisation et peut être modifié dans la définition du format des données exportées (voir Mise en forme des résultats).

Echelle de couleurs

Cette partie permet de définir la valeur qui est associée à chaque couleur du graphique. Une valeur est associé à chacune des couleurs extrêmes de l'échelle. Les valeurs intermédiaires sont calculées linéairement par projection (dans l'espace mathématique des couleurs) sur l'axe reliant ces deux couleurs.

Plus simplement, si l'une des couleurs est le noir et l'autre le blanc, les valeurs calculées pour les couleurs rencontrés correspondent à sa luminosité.

Exemples:

Couleur 1 Valeur 1 Couleur 2 Valeur 2 Pixel Valeur associée
(255,255,255) blanc 0 (0,0,0) noir 100 (127,127,127) gris 50
(0,0,0) noir 0 (255,0,0) rouge 100 (0,255,0) vert

0

(il n'y a pas de composante verte dans le rouge)

Pour modifier les couleurs de l'échelle, vous pouvez soit cliquer sur le bouton de couleur (une boite de dialogue apparaît, permettant de définir chacune de ses composantes), soit utiliser les boutons pour la choisir directement sur le graphique.

Le bouton "Utiliser par défaut" permet de sauvegarder l'échelle de couleur (couleurs et valeurs associées) afin de pouvoir utiliser cette même échelle sur plusieurs courbes de densitométrie différentes.
