Mise en forme des résultats

Avant d'enregistrer les résultats dans un fichier, il est nécessaire de mettre en forme les données précédemment identifiées.

Après la demande d'enregistrement (menu "Fichier / Exporter les données" par exemple), une boite de dialogue s'ouvre, permettant cette mise en forme. Les noms des courbes qui ont été précédemment identifiées figurent initialement dans la liste de gauche de la boite de dialogue. Pour que les données d'une courbe soient exportées, il suffit de la faire passer dans la liste de droite au moyen du bouton flèche correspondant. Les deux boutons situés à droite de la boite de dialogue permettent de faire varier l'ordre dans lequel ces courbes apparaîtront dans le fichier. Le bouton "Prévisualisation" permet d'afficher à l'écran le résultat final. Le bouton "Format" permet de choisir le caractère de séparation des colonnes dans le fichier d'export.

Pour les courbes continues, les courbes splines et les courbes de densitométrie, une boite de dialogue apparait lorsqu'on fait passer la courbe de la liste de gauche à celle de droite. Cette boite de dialogue permet de spécifier la quantité de points à enregistrer dans le fichier final.

Pour les courbes de densitométrie, il est possible de faire apparaitre dans le fichier de sortie soit la valeur d'abscisse des points calculés soit leur valeur d'ordonnée, soit les deux. Lorsque le segment analysé est parallèle à l'axe des abscisses, seules les valeurs d'abscisses sont affichables. Il en est de même lorsque le segment est parallèle à l'axe des ordonnées (l'autre coordonnée est constante).

Initialement, le premier point à enregistrer est le point du segment qui a l'abscisse la plus faible. Ce premier point doit appartenir au segment qui a été tracé et dont les coordonnées minimales et maximales sont affichées dans la boite de dialogue. Les valeurs de x et de y sont liées entre elles puisque ce point appartient au segment : le fait de modifier son abscisse modifie automatiquement son ordonnée et inversement.

Vous pouvez exprimer la fréquence des points à enregistrer soit selon les abscisses soit selon les ordonnées. Une fréquence (ou période) négative est autorisée. Tout comme pour le premier point, le fait de modifier la fréquence selon X modifie automatiquement celle selon Y. Lorsqu'un axe est logarithmique, la période est exprimée en décade d'unité.

 

Les données ainsi générées sont enregistrées dans un format directement lisible par MS-Excel ou par tout autre type de tableur.
