Open graphic

This command is available using the File / Open graphic menu or the icon of the toolbar.

Before any other operation, you must open (or scan) the graphic file which contain the curves you want to identify. CurveUnscan can open almost every existing graphic file formats : bmp, pict, tif, jpg, gif, png, ...

If a file has already been opened and some curves have been identified but not saved, a confirmation is asked.
