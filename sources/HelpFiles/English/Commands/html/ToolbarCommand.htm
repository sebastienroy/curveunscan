Toolbar

You can show or hide the toolbar using the View / Toolbar menu.

The toolbar contains the following elements:

These tools are related to the followings commands (from left to right) :

  • File / Open graphic
  • File / Data export
  • Graph construction / New curve / Continuous curve
  • Graph construction / New curve / Discrete curve
  • Graph construction / Both (new coordinates system and new curve)
  • Graph construction / New coordinates system
  • Help / About CurveUnscan

The last tool is used to display on-line help about the different commands.
