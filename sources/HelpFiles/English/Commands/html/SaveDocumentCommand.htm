Save document

By this command you can save a CurveUnscan document. It is available by the File / Save document menu. It saves the document into a CurveUnscan file format.

The CurveUnscan documents contain a graphic and all the elements necessary for the curve identification : axes, initial points, curve names ...

The file generated by this command cannot be processed by other software, contrary to the file generated by the command "Data export". However, by saving the document, you can open it later, for further curve recognition on the same graphic for instance. It is not necessary to perform the recognition from the beginning.
