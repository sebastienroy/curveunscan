Manual modifications

The previously identified curve may not be perfect, due to some annotations on the graphic for instance. In order to remove these imperfections, you can directly modify the curve on the graphic by drawing on it.

Two different tools are available for the modifications : a pen and a rubber .

When the pen is used, the draw replaces the points which have the same abscissa value.

When the rubber is used, the points which have the same abscissa value than the cursor are erased.

Undo / Redo

You can undo or redo every modification operation by using the buttons (undo) and (redo). The number of operation kept in memory is only limited by the total amount of available memory. However, the modification history is lost when this dialog box is closed (by using the "Previous" button for instance).

Reset

This button cancels all the modification by rebuilding the curve from the data previously defined. This operation can also be canceled by the "Undo" button
