Options : Densitometry curve

With this panel, you can change the default color scale values and color used for the construction of the densitometry curves.

A value is associated to each of the two colors of the scale. During the curve identifications, the values of the encountered intermediate colors are calculated linearly by projection (in the color mathematical space) on the axis that links these two colors.

In other words, if one of the colors is black and the other is white, the calculated values for the encountered colors are related to their brightness.

Examples:

Color 1 Value 1 Color 2 Value 2 Pixel Related value
(255,255,255) white 0 (0,0,0) black 100 (127,127,127) gray 50
(0,0,0) black 0 (255,0,0) red 100 (0,255,0) green

0

(there is no green component in the red)

To modify the scale colors, click on the colored button (a dialog bow appears, allowing to adjust each of the color component).
