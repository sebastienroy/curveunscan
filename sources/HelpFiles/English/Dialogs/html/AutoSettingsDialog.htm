Options : Prompt

This tab allows you to modify some automatic behaviours of CurveUnscan

Starting action

When starting, CurveUnscan can ask you for opening a graphic file, importing one from scanner or nothing at all.

Prompt for data export

After each curve identification or modification CurveUnscan asks you (or not) if you want to export the data. If your graphics are usually composed of several curves, desactivate this feature for a better comfort.
