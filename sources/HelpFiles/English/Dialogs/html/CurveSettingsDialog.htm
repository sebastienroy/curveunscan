Options : Continuous curve

With this panel, you can change the default values used for the construction of the continuous curves.

Line thickness compensation algorithm

Two different algorithms can be used for the line thickness compensation :

"basic" algorithm : moves the points of the recognized curve with a fixed value. The apparent compensation is so not the same on sharp parts of the curve than on flat ones.

"Smart" algorithm : adjusts the compensation depending on the curve derivative. This results to a better compensation, either on flat parts or on sharp ones. The curve derivative is computed around each point using a range of the neighbor points. The number of points used to compute this derivative can be adjusted from 1 to 32. High values can lead to a bad compensation due to the "inertia" of the computation, especially with complicated curves.

Threshold color

The threshold color is the color used to differentiate the points belonging to the curve from the points belonging to the background of the graphic. This color is taken into account during the identification of colored and gray scale graphics. This color is defined with each of its components : red, green, blue. Each component has a value comprised between 0 (darkest) and 255 (brightest). A pixel of the graphic is considered as belonging to the curve if each of its component values are lower than the threshold color ones.

A too dark threshold color leads to a bad curve identification if the graphic is weekly contrasted.

A too bright threshold color leads to the confusion between the "background noise" of the graphic and the curve.

In order to change the threshold color, click on the colored button.

Followed side

The curve identification is made by following the line, either the top side of the curve or the bottom side. If the curve contains sharp positive peaks, following the top leads to better results (contrarily, if the curve contains negative peaks, following the bottom is better).

Dashed curve

By specifying that the curve is dashed, its following is not stopped after each dash but is continued to the next dashes. The maximal distance between dashes can also be changed.

Default values

When this button is clicked, all the preview values are set to their defaults.
