First point and saving frequency for continuous curves

First value

This is the value of the first point that will be written in the output file.

Frequency-period

Enter the frequency of the saved points or the period between each of them.

Each of this two fields is updated according to the other. Frequency and period must be greater than zero.

Units

The unit that appears in front of each field is that previously defined for the X axis and will appear too in the output file. In the case of a logarithmic scale, the unit is decade of X.
