Curve limits

You must specify the limits into which the curve will be recognized.

You can enter this value either "by-hand" or by clicking on the "Pick position" button then on the desired location on the graphic. The related field will be filled with the abscissa value corresponding to this point.

In the case of a direct axis (right scale point greater than left one), the right limit value must be greater than the left one. In the case of a reversed axis (left scale point value greater than right one), left limit must be greater than the right one.

If you currently modify an already constructed curve, changing the limits have the following consequences :

  • All hand-modifications are lost.
  • If an initial point is outside the new limits, it will be removed (all the curve is able to disappear).
