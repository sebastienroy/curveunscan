Options : Export

With this tab, you can change the default export file format.

Column separator

In the exported files, the data related to each curve, and their X and Y values, are separated with a special character. Using tab character allow this file to be directly recognized by MS-Excel for instance.

Default extension

The filename extension has no influence on its format. However, this extension allows the files to be directly recognized by some softwares, which can open them using the appropriate manner.
