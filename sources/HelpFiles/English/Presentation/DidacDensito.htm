Densitometry curve

This kind of curve is used to analyse a color profile on the graphic. A value is associated to each pixel of a segment (user-defined) on the graphic. This value is calculated according to the pixel color and a color scale (right part of the dialogbox)

Draw segment

When the segment to analyze has not already been drawn, the "Draw segment" button is pressed. So draw a segment on the part of the graphic you want to analyze. If this segment is larger than the window, the graphic scrolls automatically. When the "Shift" key is pressed, the line is drawn parallel to the border of the window.

Preview

When the segment has been defined, the "Preview" button can be used to show the color profile on this segment. The direction of this profiles goes from the beginning of the segment (left side of the preview window) to its end. However, this representation direction is only used for the preview and can be modified during the exported data format definition (see Results formating )

Color scale

This part of the dialog box is used to define the value associated to each color of the graphic. A value is associated to each of the two colors of the scale. During the curve identification, the values of the encountered intermediate colors are calculated linearly by projection (in the color mathematical space) on the axis that links these two colors.

In other words, if one of the colors is black and the other is white, the calculated values for the encountered colors are related to their brightness.

To modify the scale colors, you can either click on the colored button (a dialog bow appears, allowing to adjust each of the color component), or use the buttons to directly choose the color on the graphic.

The "Use as default" button is used to save the color scale (colors and related values) in order to use the same color scale with different densitometry curves.
