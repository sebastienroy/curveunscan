Curves identification

When starting CurveUnscan, the software automatically asks you, depending on the selected option (see View / Options...), either to scan a new graphic or to open one from a file.

Once the graphic imported, the curve identification sequence begins automatically.

This sequence is made up of several steps:

  • Coordinates system identification
    • X axis and scale identification
    • Y axis and scale identification
  • Choice of the kind of curve to identify
    • Discrete curve identification
    • Continuous curve identification:
      • Definition range identification
      • Curve location identification
      • Manual correction
    • Spline curve identification
    • Densitometry curve identification
  • Data formatting and export
