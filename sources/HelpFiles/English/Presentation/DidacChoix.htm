Choice of the kind of curve

Once the coordinates system is defined, you have to choose the mode of the curve analysis.

CurveUnscan performs recognition using four different modes: continuous curves, spline curves, discrete curves (scattered points), and densitometry curves.

The continuous curves are recognized from different element chosen by the user : definition range, location on the graphic, line thickness, ...

The spline mode allows to analyse the graphic using spline curves. These mathematical curves are build as in drawing tool, from "anchor points" chosen by the user.

The discrete curves, or scattered points, are made of a set of unrelated points. These points are individually identified by the user.

The densitometry curve consist to the analysis of a color profile, on a segment of the graphic.
