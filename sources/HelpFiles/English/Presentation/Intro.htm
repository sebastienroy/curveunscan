How does CurveUnscan Work?

The curve recognition process consists of three main steps :

  • Coordinates system recognition
  • Curve Recognition
  • Data formatting and export.

Each of these steps deal with independent data. This allows the user to modify the data concerning each step without impacting an other one.

One or more coordinates systems can be built on a same graphic. One or more curves can be built on each coordinates system. The number of coordinates systems or curves that can be built is only limited by the available memory.

The information needed by each recognition step are entered using dialog boxes gathered into sequences. Each sequence allow the user to build or to modify one kind of object, coordinates system or curve.

The object that has been constructed can be selected using the double clicks (the selected object is then displayed bold). The commands (modification, deletion, ...) are then applied on the selected object.
