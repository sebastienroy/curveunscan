Spline curve

The spline curves allows to digitize graphics very easily, by drawing on the original curves, as in a drawing tool. The spline curves are drawn from the anchor points choosen by the user. The anchor points are either fixed points or slope points.

The creation and the modification of the spline curves are made through 5 different tools (buttons from left to right) :

  • Adding points at the end of the curve (default selected tool)
  • Adding points at the beginning of the curve
  • Adding points in the middle of the curve
  • Points removal
  • Points modifications

Adding points at the end of the curve

With this tool, the fixed points are created at the place where the left mouse button is pressed. The assocaited slope points are created where the mouse button is released. When the slope point is dragged, another slope point is drawn on the other side of the fixed point, symmetrically. If the Ctrl key is pressed, the two slope points are unbinded. This behaviour can be used to make sharp corners. The Shift key can be used to re-bind them, and then, smooth the corners. Each new point is created at the end of the cure.

Adding points at the beginning of the curve

This tool is similar to the previous one. The new points are created at the other end of the curve.

Adding points in the middle of the curve

With this tool, it is possible to add points in the middle of allready build curves. The slope points are adjusted in order to have exactely the same curve path with the new point. Once a new point has been added, the "Points modification" tool is automatically activated.

Points removal

With this tool, the user can remove points created previously.

Points modifications

This tool allows to modify (position or slope) the points of a spline curve.
